# Support

Use GitHub Issues for bug reports, documentation problems, and feature requests.

Before opening an issue:

- Check the README and existing issues.
- Confirm whether the behavior comes from Mapshaper.Net or the external mapshaper CLI.
- Include the Mapshaper.Net version, mapshaper CLI version, operating system, and .NET version.

For questions about mapshaper command behavior, see the upstream mapshaper project:

https://github.com/mbloch/mapshaper
