# Documentation

This directory contains GitHub-compatible Markdown documentation for Mapshaper.Net. The pages are written so they can be read directly in the repository or copied into a GitHub Wiki.

## Pages

- [Home](Home.md)
- [Getting Started](Getting-Started.md)
- [Supported Functions](Supported-Functions.md)

## Maintainer notes

- Use standard GitHub Flavored Markdown.
- Keep one topic per page.
- Use relative links between documentation pages.
- Use absolute repository links when linking from wiki pages back to top-level repository files.
- Use title-case filenames with hyphens, such as `Supported-Functions.md`, for wiki-compatible page names.
- Keep code examples complete enough to copy into a small C# project.
